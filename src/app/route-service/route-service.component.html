Route State

Route state refers to global state that is scoped to a specific route. It becomes accessible only when within the route where the associated service is provided. This is typically achieved by providing the service directly in the route configuration. An instance of the service is created the first time the route is visited, and it persists across future visitsâ€”meaning it won't be destroyed or recreated on subsequent navigations to the same route.
